# 亲友试用发送话术

这个文档用于把 Mama Save JP 发给 5-10 个亲友试用。当前阶段只做人工邀请和人工记录，不做表单、登录、订阅或追踪链接。

## 使用前检查

发送前先确认：

- 本周没有明显过期或无法解释来源的优惠。
- 只引用当前站内已核验且仍有效的信息。
- 文案不承诺“必省钱”“最低价”“一定返”，只说明可以帮忙判断。
- 每次只请对方完成 3 个小任务，避免像正式问卷。
- 收到反馈后记录原话，再填入 `docs/friend-feedback-checklist.md` 的模板。

## 微信或 LINE 私聊版

```text
我最近在整理一个给在日华人宝妈/宝爸用的小站，主要是把日本母婴用品怎么买、什么时候买、哪些活动要注意，用中文整理出来。

想请你帮我试用 5 分钟，不用认真看完整站，只做 3 件事：

1. 打开首页后，告诉我 30 秒内能不能看懂它是做什么的。
2. 从“本周值得买 / 省钱日历 / 新手攻略”里点一个你最可能用的入口。
3. 找一个让你觉得有帮助、或让你不信任的地方，直接截图或发原话给我。

这个站不需要登录，也不会收集你的信息。现在还在早期，我最想知道的是：它能不能帮你少踩坑，以及哪里看起来不够可信。
```

## 跟进追问

对方回复后，只追问 1-2 个最关键问题：

1. 你下次会在什么场景再打开它？
2. 哪个页面最值得发给同在日本的朋友？
3. 哪一句话或哪个字段让你担心信息不准？
4. 你现在最想补的主题是纸尿裤、湿巾、辅食、洗护、保育园、童装，还是平台规则？

## 反馈处理规则

- 如果反馈指向过期、来源不清或价格表达过度确定，优先修信任问题。
- 如果 3 人以上都找不到入口，再调整首页或导航结构。
- 如果只是单个主题缺内容，先记录到下一轮攻略或优惠数据任务。
- 不因为 1 条反馈就做大范围改版。
