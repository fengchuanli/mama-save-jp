# 小红书/朋友圈引流文案库

这个文档用于沉淀 Mama Save JP 早期增长文案。当前阶段不做表单、付费群和复杂追踪，先用清晰、克制、可转发的内容验证在日华人宝妈是否愿意收藏和回访。

## 使用原则

1. 不夸大优惠，不写“全网最低”“必买”“闭眼入”。
2. 每条文案只承诺一个明确价值：少踩坑、看懂规则、知道什么时候买。
3. 引导用户打开一个具体页面，不把所有入口都塞进同一条文案。
4. 优先面向 0-3 岁宝宝家庭、刚来日本的新手宝妈和准备保育园用品的家庭。
5. 发布前根据当周真实日历和优惠状态，删掉不确定、已过期或无法核验的信息。

## 文案清单

### 1. 新手准备清单

- 标题：刚来日本准备宝宝用品，先别急着囤一堆
- 正文：很多母婴用品不是越早买越好，尤其纸尿裤、童装和保育园用品，买早了容易尺码不合或活动没赶上。我整理了一个日本母婴省钱小站，先从 0-3 岁宝宝常买品类、购买节奏和新手攻略看起，适合刚开始做清单的宝妈。
- 引导页：`/guides/newborn-shopping-list`

### 2. 尿不湿补货

- 标题：在日本买纸尿裤，先看单片价再看折扣
- 正文：日本平台经常写积分、优惠券、组合装，看起来都很划算，但真正要比的是单片价、尺码和宝宝当前用量。我把尿不湿怎么买、什么情况别囤、怎么判断好价线整理成中文攻略，补货前可以先看一眼。
- 引导页：`/guides/buy-diapers-japan`

### 3. 楽天规则解释

- 标题：楽天 5 と 0 のつく日，到底适不适合买母婴用品？
- 正文：楽天活动多，但不是每次都适合下单。要看报名、支付方式、积分上限、券后价格和是否刚好需要补货。我整理了新手能看懂的中文步骤，适合准备买纸尿裤、湿巾、辅食和洗护补充装前快速核对。
- 引导页：`/guides/rakuten-5-0-mama-shopping`

### 4. 省钱日历收藏

- 标题：在日本养娃，母婴用品可以按活动节奏买
- 正文：纸尿裤、湿巾、童装、洗护和保育园用品，不一定要看到折扣就下单。可以先看近期 Amazon、楽天、西松屋、赤ちゃん本舗、药妆店有哪些活动，再决定是当天买、提前准备，还是继续观望。
- 引导页：`/calendar`

### 5. 保育园准备

- 标题：准备保育园用品，别一次买太多同款
- 正文：保育园用品很容易越买越多，但每个园要求不一样，孩子尺码也变得快。建议先看入园说明，再按备用衣、午睡用品、餐具和姓名贴慢慢补。我整理了一篇中文清单，适合第一次准备保育园的家庭。
- 引导页：`/guides/nursery-entry-budget-items`

### 6. 西松屋/母婴店选择

- 标题：西松屋、赤ちゃん本舗、药妆店，母婴用品到底去哪买？
- 正文：不同店适合买的东西不一样。童装、日用品、婴儿车、辅食、洗护和应急补货，价格和体验都不同。我整理了在日本常见母婴购买渠道的差异，适合刚开始比较平台的新手宝妈。
- 引导页：`/guides/choose-baby-stores-japan`

### 7. 童装尺码

- 标题：日本童装 80/90/100 码怎么选，别只看年龄
- 正文：日本童装尺码看似简单，但每个孩子身高体重、季节和品牌版型都不同。换季买衣服前，最好先看现有衣服还能穿多久，再决定要不要提前囤。我整理了 80/90/100 码的简单判断方式。
- 引导页：`/guides/kids-clothes-size-80-90-100`

### 8. 药妆店优惠券

- 标题：药妆店优惠券买母婴用品，先确认対象商品
- 正文：药妆店优惠券看起来很好用，但纸尿裤、奶粉、辅食和指定品牌经常有対象外条件。下单前先看券是否能用、是否比电商箱购便宜、是否急用。我整理了药妆店买母婴用品的避坑点。
- 引导页：`/guides/drugstore-coupon`

### 9. 本周值得买

- 标题：看到母婴优惠先别冲，先问 3 个问题
- 正文：家里真的快用完了吗？券后单价比平时低吗？这个尺码/型号还能用多久？我把适合买和不适合买的情况写进了优惠卡片，适合下单前快速判断，不用在群消息里翻半天。
- 引导页：`/deals`

### 10. 给群友转发

- 标题：群里常问的日本母婴省钱问题，我做了一个中文入口
- 正文：什么时候买尿不湿、楽天活动怎么看、西松屋适合买什么、保育园用品怎么准备，这些问题反复解释很耗时间。我把攻略、省钱日历和本周值得买放在一个静态小站里，适合收藏后慢慢查。
- 引导页：`/`

## 发布节奏建议

- 周一或活动前：发省钱日历收藏文案，提醒用户先看近期节点。
- 平台活动当天：发楽天规则或本周值得买文案，但只引用当前已确认的活动。
- 开学/入园季：发保育园准备、童装尺码和母婴店选择文案。
- 群聊答疑后：发“给群友转发”文案，承接重复问题。
