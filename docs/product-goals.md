# Mama Save JP 产品目标

## 产品定位

Mama Save JP 是面向在日华人宝妈/宝爸的日本母婴省钱信息站，帮助 0-3 岁宝宝家庭理解日本母婴用品什么时候买、在哪里买、怎么买更划算。

第一阶段不做登录、不做数据库、不做爬虫，先通过静态内容、人工精选数据和定时优化，验证用户是否需要这类信息。

## 1 年内目标

1. 成为在日华人宝妈查询日本母婴省钱信息的轻量入口。
2. 覆盖 0-3 岁家庭最常买的品类：尿不湿、湿巾、童装、保育园用品、辅食、洗护、妈妈用品。
3. 形成可持续更新的内容结构：优惠列表、省钱日历、攻略文章、平台规则解释。
4. 通过 SEO 和小红书/微信群分享获得第一批真实访问用户。
5. 验证变现方向：返佣链接、资料包、微信群精选提醒、品牌合作前的用户信任。

## 1 年内关键指标

- 攻略文章达到 50 篇以上。
- 省钱日历覆盖 8-12 个常用平台/店铺。
- 优惠数据保持每周至少 3 次更新。
- 每月自然访问达到 1,000-5,000。
- 收集 30 条以上真实宝妈反馈。

## 2-5 年中期目标

1. 从静态信息站升级为日本母婴省钱资料库。
2. 增加按月龄、品类、平台、保育园场景的筛选和搜索。
3. 建立日本母婴常用品价格档案和好价线。
4. 扩展到邮件/LINE/微信群提醒，做轻量订阅服务。
5. 形成可迁移能力：日本生活省钱、育儿用品选购、华人家庭本地化信息服务。

## 2-5 年可探索方向

- 日本母婴用品好价数据库。
- 0-3 岁月龄购物清单。
- 保育园入园准备清单工具。
- 在日华人宝妈省钱社群。
- 母婴返佣和品牌合作，但必须坚持真实、谨慎、不夸大优惠。

## 产品原则

1. 不夸大优惠，不制造焦虑。
2. 优先解释“为什么值得买”，而不是只堆链接。
3. 面向新手宝妈，用简单中文解释日本平台规则。
4. 健康、安全、医疗相关内容只做信息整理，不替代专业建议。
5. 先小步快跑，避免过早做复杂功能。
