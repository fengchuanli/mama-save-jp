# Mama Save JP 产品目标

## 产品定位

Mama Save JP 是面向在日华人宝妈/宝爸的日本母婴省钱信息站，帮助 0-3 岁宝宝家庭理解日本母婴用品什么时候买、在哪里买、怎么买更划算。

第一阶段不做登录、不做数据库、不做爬虫，先通过静态内容、人工精选数据和定时优化，验证用户是否需要这类信息。

数据来源、优惠核验和自动化边界以 `docs/compliance-rules.md` 为准。产品增长不能以绕过平台限制、批量抓取或误导用户为代价。

## 1 年内目标

1. 成为在日华人宝妈查询日本母婴省钱信息的轻量入口。
2. 覆盖 0-3 岁家庭最常买的品类：尿不湿、湿巾、童装、保育园用品、辅食、洗护、妈妈用品。
3. 形成可持续更新的内容结构：优惠列表、省钱日历、攻略文章、平台规则解释。
4. 通过 SEO 和小红书/微信群分享获得第一批真实访问用户。
5. 验证变现方向：返佣链接、资料包、微信群精选提醒、品牌合作前的用户信任。

## 1 年内关键指标

- 攻略文章达到 50 篇以上。
- 省钱日历覆盖 8-12 个常用平台/店铺。
- 优惠数据保持每周至少 3 次更新。
- 每月自然访问达到 1,000-5,000。
- 收集 30 条以上真实宝妈反馈。

## 2-5 年中期目标

1. 从静态信息站升级为日本母婴省钱资料库。
2. 增加按月龄、品类、平台、保育园场景的筛选和搜索。
3. 建立日本母婴常用品价格档案和好价线。
4. 引入个人账号功能，让用户可以用 email 和密码登录，保存家庭常用品、购买频率、宝宝月龄和常用平台。
5. 记录用户通过本站决策带来的估算省钱金额，例如每月节省多少钱、哪些品类最值得关注，增强订阅动力。
6. 扩展到邮件/LINE/微信群提醒，做轻量订阅服务。
7. 形成可迁移能力：日本生活省钱、育儿用品选购、华人家庭本地化信息服务。

## 2-5 年可探索方向

- 日本母婴用品好价数据库。
- 0-3 岁月龄购物清单。
- 保育园入园准备清单工具。
- 在日华人宝妈省钱社群。
- 个人账号、购买频率记录、家庭常用品清单和月度省钱报告。
- 基于用户购买频率的个性化提醒，例如尿不湿快用完、湿巾低于好价线、保育园用品换季补货。
- 母婴返佣和品牌合作，但必须坚持真实、谨慎、不夸大优惠。

## 产品原则

1. 不夸大优惠，不制造焦虑。
2. 优先解释“为什么值得买”，而不是只堆链接。
3. 面向新手宝妈，用简单中文解释日本平台规则。
4. 健康、安全、医疗相关内容只做信息整理，不替代专业建议。
5. 先小步快跑，避免过早做复杂功能。
6. 个人账号和省钱记录涉及隐私与安全，必须在站点内容、真实优惠数据和订阅价值被验证后再实现；不在早期静态信息站阶段贸然收集用户密码和家庭消费数据。
7. 任何优惠、价格、库存、积分、自动化相关工作，都必须遵守 `docs/compliance-rules.md`；不模拟登录、不绕过反爬、不批量抓取、不把未核实信息写成确定优惠。
